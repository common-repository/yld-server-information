== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in 
WordPress



== Frequently Asked Questions ==
= Where can I see the information? =
After active the plugin, you will find an item "Server Info" in admin menu, click on it, it will show all information in a page.
= What information does it show? =
System, Server Environment, available space, Server name, IP, port number, document root, mysql version, php version, php max upload filesize, post size...
